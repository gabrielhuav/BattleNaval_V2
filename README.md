# BattleNaval V2 (BitBattles)

**BitBattles** es una aplicación móvil educativa para Android, desarrollada en Kotlin, que combina el clásico juego de "Batalla Naval" con conceptos fundamentales de computación. El objetivo es enseñar sobre el **sistema binario**, el **código ASCII** y las **unidades de medida de almacenamiento** de una manera lúdica e interactiva.

Este proyecto fue desarrollado como parte de la Maestría en Docencia para la Educación Media Superior (MADEMS) de la UNAM y fue implementado como una herramienta de enseñanza en el Taller de Cómputo del CCH Naucalpan.

## Descripción del Proyecto

La aplicación surge de la necesidad de encontrar métodos de enseñanza más atractivos para conceptos abstractos de la informática. "BitBattles" transforma el aprendizaje en una experiencia interactiva a través de dos modalidades principales:

* **Sección Educativa:** Módulos interactivos para aprender sobre el sistema binario, el código ASCII y las unidades de almacenamiento.
* **Sección de Juegos:** Incluye "Hundir la Flota Digital" (BitBattles), donde los jugadores deben convertir coordenadas a sistema binario y ASCII para poder jugar.

## Características Principales

### Educativas
* **Aprendizaje Interactivo:** Tutoriales sobre el sistema de numeración binario, decimal y código ASCII.
* **Conversor ASCII:** Una herramienta para convertir texto a su representación decimal y binaria en tiempo real.
* **Unidades de Medida:** Explicaciones sobre las unidades de almacenamiento digital como bytes, kilobytes, megabytes, etc.

### Lúdicas
* **Batalla Naval Binaria (BitBattles):** Juega al clásico "Hundir la Flota" donde las coordenadas se deben descifrar usando conversiones binarias y ASCII.
* **Modos de Juego:**
    * **Un Jugador:** Reta a la IA y pon a prueba tus conocimientos.
    * **Multijugador por Bluetooth:** Juega con un amigo en tiempo real.
* **Minijuegos Educativos:** Desafíos de conversión de decimal a binario y viceversa para practicar tus habilidades.

## Tecnologías Utilizadas

* **Lenguaje:** Kotlin
* **Plataforma:** Android
* **Entorno de Desarrollo:** Android Studio
* **Comunicación:** Bluetooth para el modo multijugador

## Instalación

Para probar la aplicación:

1.  Clona este repositorio:
    ```
    git clone [https://github.com/gabrielhuav/BattleNaval_V2.git](https://github.com/gabrielhuav/BattleNaval_V2.git)
    ```
2.  Abre el proyecto en Android Studio.
3.  Compila y ejecuta la aplicación en un emulador o dispositivo Android.
